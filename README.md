### API

``` js
const gnr = require("./release");

const gnr.release(gnr.Type.MINOR);
```

### CLI

``` console
./release.cli.js <arg|shorthand>
```

| Release Type | `arg` | `shorthand` |
| :- | :- | :- |
| Major |`--major` | `-M` |
| Minor | `--minor` | `-m` |
| Patch | `--patch` | `-p` |
